### Scripts Implemented

All 8 scripts are available in `nix develop`:

- **`garage-dev-start`** — Starts persistent Garage on ports 3900-3903. First run generates secrets, writes `garage.toml`, bootstraps layout/bucket/key. Handles stale PIDs, port conflicts, concurrent starts via `flock`.
- **`garage-dev-stop`** — Graceful SIGTERM with 10s timeout then SIGKILL.
- **`garage-dev-status`** — Reports running/stopped with health check.
- **`garage-dev-info`** — Prints credentials; `--env` flag for `eval $(...)` usage.
- **`garage-test-start`** — Ephemeral instance on ports 3910-3913 with deterministic secrets, SQLite backend, fresh temp directory each time.
- **`garage-test-stop`** — Stops and `rm -rf`s the entire temp directory.
- **`garage-test-status`** — Reports running/stopped for test instance.
- **`garage-test-info`** — Same as dev-info but for test instance.
